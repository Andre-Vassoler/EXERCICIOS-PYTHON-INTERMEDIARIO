# EXERCICIOS-PYTHON-INTERMEDIARIO
1. **1_Funcao.py**  
   Introdução à criação e utilização de funções em Python.

2. **1_Funcao2.py**  
   Continuação dos estudos sobre funções, explorando parâmetros e retorno de valores.

3. **2_Perguntas_Respostas.py**  
   Sistema simples de perguntas e respostas utilizando listas e condicionais.

4. **2_Perguntas_Respostas2_zip.py**  
   Implementação do sistema de perguntas utilizando a função `zip` para percorrer múltiplas listas.

5. **2_Perguntas_Respostas3_funcao.py**  
   Refatoração do sistema de perguntas aplicando funções para organizar melhor o código.

6. **2_Perguntas_Respostas4_funcao.py**  
   Versão aprimorada com maior modularização utilizando funções.

7. **2_Perguntas_Respostas5_enumerate.py**  
   Uso da função `enumerate` para numerar perguntas e melhorar a exibição ao usuário.

8. **3_Lista_dentro_Lista.py**  
   Manipulação de listas aninhadas (listas dentro de listas).

9. **3_Lista_dentro_Lista2.py**  
   Continuação do uso de listas aninhadas com operações mais avançadas.

10. **4_Lista_com_Dicionarios.py**  
    Estruturação de dados utilizando listas contendo dicionários.

11. **5_Unir_Listas.py**  
    Métodos para unir listas em Python.

12. **5_Unir_Listas2_zip.py**  
    União de listas utilizando a função `zip`.

13. **5_Unir_Listas3_itertools_zip.py**  
    Uso de ferramentas da biblioteca `itertools` para combinar listas.

14. **6_Somando_Listas_zipper.py**  
    Soma de elementos de listas utilizando emparelhamento com `zip`.

15. **6_Somando_Listas2_range.py**  
    Soma de listas utilizando índices com `range`.

16. **6_Somando_Listas3_enumerate.py**  
    Soma de listas utilizando `enumerate` para acesso aos índices.

17. **6_Somando_Listas4_list_comprehension.py**  
    Implementação da soma de listas utilizando list comprehension.

18. **6_Somando_Listas5_Explicacao.py**  
    Explicação detalhada das diferentes formas de somar listas em Python.

19. **7_Lista_Tarefas_Desfazer_Refazer2.py**  
    Sistema de lista de tarefas com funcionalidades de desfazer e refazer ações, simulando um gerenciador simples de tarefas.

20. **Exercicio5_package/**  
    Exemplo de organização de código em pacotes (packages), demonstrando modularização e importação entre arquivos.
